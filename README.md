# Mall Queen Web Mobile Deploy

This repository contains the static Cocos Creator `web-mobile` build for Render.

Source build path:

```text
/Users/julian/Project/mall-queen/mall-queen-cocos/build/web-mobile
```

Render configuration:

```text
Service type: Static Site
Name: mall-queen-web-mobile
Build command: echo "Static Cocos build already committed"
Publish directory: .
```

No paid Render plan is required for this static deployment.
